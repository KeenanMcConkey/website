# website
Website for Keenan McConkey

## About
This is a personal website hosted on GitHub Pages with a custom domain at [keenanmcconkey.dev](https://keenanmcconkey.dev).

## Features
- Modern, responsive design
- Mobile-friendly layout
- Smooth scrolling navigation
- Professional sections: About, Skills, Projects, Contact
- Custom domain support
- Easy to customize

## Local Development
To view the website locally:
1. Clone this repository
2. Open `index.html` in your web browser

## Customization
- Edit `index.html` to update content
- Modify `styles.css` to change the design
- Update `script.js` for additional interactivity

## Deployment
This website is automatically deployed via GitHub Pages. Any changes pushed to the main branch will be reflected on the live site.

## Custom Domain
The custom domain `keenanmcconkey.dev` is configured via the `CNAME` file. Make sure your domain's DNS settings point to GitHub Pages:
- Create an A record pointing to GitHub Pages IPs
- Or create a CNAME record pointing to your GitHub Pages URL

For more information, see [GitHub Pages documentation](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site).
